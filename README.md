Architected a multi-building campus network for Sindh Madressatul Islam University supporting over 450 devices. The project involved end-to-end network planning and implementation, including custom IP subnetting via VLSM, VLAN segmentation, and Single-Area OSPF routing. Secured the network infrastructure by configuring Access Control Lists (ACLs) for granular resource access, implementing Port Security, setting up NAT for external communication, and establishing secure remote management across all switches and routers.
